# SaaS Admin Dashboard Template

> A premium dark-themed admin dashboard template for SaaS products. Single HTML file, zero dependencies.

## Features

- ✅ Collapsible sidebar navigation
- ✅ Stats cards with trend indicators
- ✅ Revenue chart placeholder (replace with Chart.js/ApexCharts)
- ✅ Traffic source donut chart
- ✅ User data table with status badges
- ✅ Search bar header
- ✅ Notification bell
- ✅ User profile section
- ✅ Fully responsive (mobile sidebar toggle)
- ✅ CSS variables for easy theming

## Sections

- Sidebar with navigation groups
- Header with search and notifications
- 4 stat cards (revenue, users, links, conversion)
- Revenue bar chart
- Traffic donut chart
- Recent users table

## Customization

### Colors
Edit CSS variables at the top of `<style>`:
```css
--primary: #6366f1;     /* Main brand color */
--accent: #a78bfa;      /* Accent color */
--success: #22c55e;     /* Positive values */
--error: #ef4444;       /* Negative values */
```

### Data
Replace the static HTML in the table and stats with dynamic data from your API.

### Charts
Replace `.chart-placeholder` and `.donut-placeholder` with real chart libraries:
- [Chart.js](https://chartjs.org)
- [ApexCharts](https://apexcharts.com)
- [Recharts](https://recharts.org) (React)
