## How to Customize

### 1. Colors
Edit the CSS variables at the top of the `<style>` tag:
```css
:root {
  --brand: #6366f1;      /* Primary color */
  --brand-light: #818cf8; /* Light variant */
  --brand-dark: #4f46e5;  /* Dark variant */
  --accent: #a78bfa;      /* Accent color */
}
```

### 2. Content
Replace placeholder text in the HTML:
- Company name: Search for "LaunchWave"
- Hero text: Edit the `<h1>` and `<p>` in `.hero`
- Features: Update `.feature-card` elements
- Pricing: Modify `.pricing-card` sections
- Testimonials: Replace with real testimonials

### 3. Images
Replace the `.mockup-placeholder` div with your actual product screenshot:
```html
<div class="mockup-content">
  <img src="your-screenshot.png" alt="Product Screenshot" />
</div>
```
