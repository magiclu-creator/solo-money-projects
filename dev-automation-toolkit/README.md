## Quick Start

```bash
# Scaffolding a new project
node scripts/scaffold.js nextjs my-saas-app
node scripts/scaffold.js react my-frontend
node scripts/scaffold.js python-fastapi my-api

# Smart git workflow
node scripts/git-auto.js --push
node scripts/git-auto.js --push --pr --message "feat: add user auth"

# Generate CRUD API from schema
node scripts/api-gen.js schema.json
```
