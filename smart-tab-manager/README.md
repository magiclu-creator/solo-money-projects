# Smart Tab Manager — Chrome Extension

> Organize, group, and manage your browser tabs intelligently.

## Features

- **Auto-Group**: Automatically categorizes tabs into groups (Social, Dev, AI, etc.)
- **Duplicate Detection**: Find and close duplicate tabs
- **Workspace Save/Load**: Save sets of tabs, restore them later
- **Memory Saver**: Auto-suspend inactive tabs (Pro)
- **Keyboard Shortcuts**: Quick actions from the popup
- **Dark Theme**: Beautiful, easy on the eyes

## Installation (Development)

1. Open Chrome → `chrome://extensions/`
2. Enable "Developer mode" (top right)
3. Click "Load unpacked"
4. Select the `smart-tab-manager` folder

## How to Use

1. Click the extension icon in toolbar
2. **Tabs tab**: See all open tabs, close duplicates, auto-group
3. **Workspaces tab**: Save and restore tab sets
4. **Settings tab**: Configure auto-grouping, suspending
